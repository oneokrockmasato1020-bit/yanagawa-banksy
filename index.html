YANAGAWA BANKSY は、福岡県柳川市三橋町に誕生した新感覚の MIX BAR（ミックスバー）。 可愛いキャストと飲み放題で気軽に過ごせる、柳川初のラグジュアリーな夜の遊び場。 西鉄柳川駅から車で約10分、ピンクの灯りが目印の三橋町の隠れ家です。

お一人様も、仲間と、デートで、女子会・二次会・接待まで。
男女どちらも歓迎、リーズナブルな料金で、誰もが主役になれる空間。
カラオケ・シャンパン・カクテル・焼酎多数、ここでしか味わえない特別な夜をぜひ。

— FAQ —

よくあるご質問

FREQUENTLY ASKED QUESTIONS

Q 柳川で夜遊びできるお店を探しています。 +

福岡県柳川市三橋町の YANAGAWA BANKSY（柳川バンクシー） へぜひお越しください。PM21時〜LAST（水曜定休）、可愛いキャストとお酒を楽しめる大人のMIX BARです。

Q 料金システムはどうなっていますか？ +

セット60分が 男性¥2,900 / 女性¥2,000（飲み放題込み）、延長30分¥1,500、指名料¥1,000、カラオケ1曲¥200となっております。
+¥1,000で生ビール・ハイボールも飲み放題に。TAX 10%別途／カード決済OK・手数料無料。

Q 曜日ごとの特典はありますか？ +
  • 月曜 ─ 生ビール・ハイボール飲み放題
  • 火曜 ─ カラオケ無料
  • 木曜 ─ 23時までのご来店で ¥1,000 OFF
  • 日曜 ─ 最初の1セットが 80分（通常60分）
Q 場所と営業時間を教えてください。 +

福岡県柳川市三橋町24-3 1階。柳川駅から車で約10分、ピンクの灯りが目印です。
営業時間は PM 21:00 〜 LAST、定休日は 水曜日。

Q 女性も入店できますか？ +

もちろん歓迎です。レディース料金は 60分 ¥2,000 と男性料金よりお得にご利用いただけます。女性同士のご利用、女性のお一人様も大歓迎です。

Q 予約はできますか？ +

お電話または公式SNS（Instagram / X / LINE）のダイレクトメッセージにて承っております。団体でのご来店、お祝い事、シャンパンタワーをお考えの方は、事前のご連絡をおすすめいたします。

Q ボトルキープはできますか？ +

当店ではボトルキープのご用意はございません。その代わりに、焼酎（芋・麦・米・甲類）・カクテル・サワー・ソフトドリンクなど豊富なメニューを飲み放題でお楽しみいただけます。

Q 遠方からキャストにプレゼントを贈れますか？ +

BASE 公式オンラインショップから、お気に入りのキャストにドリンクやシャンパンをリモートでプレゼントできます。離れていても想いを届けられる、新感覚のサービスです。ご注文時の備考欄に「指名キャスト名」をご記入ください。詳細は REMOTE GIFT セクションをご覧ください。

Q 大牟田・久留米・みやま市から行きやすいですか？ +

車でのアクセスが便利です。大牟田市から約20分、みやま市から約15分、大川市から約20分、筑後市から約25分、八女市から約30分、久留米市から約35分、熊本県荒尾市から約30分。福岡県南部の各市から気軽にお越しいただけます。
